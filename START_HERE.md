## ✏️ Customization

### Change Colors
See `CUSTOMIZATION_TIPS.md` for detailed instructions.

Quick find/replace:
- `from-emerald-600 to-teal-600` → Your colors

Edit component files in `/components` directory:
- Hero text → `components/Hero.tsx`
- Features → `components/Features.tsx`
- Testimonials → `components/Testimonials.tsx`
- etc.

### Add Your Logo
Replace Apple icon in `components/Navbar.tsx` with your logo.

### Add App Screenshots
Replace phone mockup in `components/Hero.tsx` with real screenshots.

## 🚀 Deployment

### Deploy to Vercel (Recommended)

1. Push code to GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin <your-repo-url>
git push -u origin main
```

2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

Done! Your site is live in ~2 minutes.

See `DEPLOYMENT.md` for more details.
