## 🚀 Getting Started

1. **Navigate to project:**
   ```bash
   cd food-doctor-landing
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run development server:**
   ```bash
   npm run dev
   ```

4. **Open browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🎨 Customization Guide

### Change Colors
Edit gradient colors in components:
```tsx
from-emerald-600 to-teal-600  // Primary gradient
from-teal-500 to-cyan-500     // Secondary gradient
```

### Update Content
All content is in component files - simply edit the text, stats, or testimonials directly.

### Add Images
Replace placeholder phone mockup in `Hero.tsx` with actual app screenshots.

### Modify Animations
Adjust Framer Motion props in components or add new animations in `tailwind.config.ts`.
